# Suporte

## Canais de suporte

| Canal | Uso |
|-------|-----|
| [Issues do GitHub](https://github.com/NullSablex/PawnPro-Engine/issues) | Bugs, comportamento incorreto, diagnósticos errados |
| [Discussions](https://github.com/NullSablex/PawnPro-Engine/discussions) | Dúvidas sobre o protocolo LSP, integração com outros editores |

> Para dúvidas sobre a extensão VS Code, use o repositório [PawnPro](https://github.com/NullSablex/PawnPro/issues).

## Ao abrir uma issue de bug

Inclua:
- Versão do motor (tag do release ou hash do commit)
- Sistema operacional e arquitetura (`uname -m` no Linux/macOS)
- Exemplo mínimo de arquivo `.pwn` que reproduz o problema
- Diagnóstico incorreto ou ausente observado

## Vulnerabilidades de segurança

Não abra issues públicas para vulnerabilidades. Consulte [SECURITY.md](SECURITY.md).
